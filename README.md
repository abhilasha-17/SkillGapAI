# SkillGapAI – AI Resume Analyzer

SkillGapAI is an AI-powered resume analysis tool that compares a candidate’s resume with a job description to identify **skill matches, missing skills, and overall alignment**.  
The system helps users understand their **skill gaps** and improve their chances of getting shortlisted for a job.


## Features

- Upload resume in **PDF format**
- Extract text from resume automatically
- Compare resume with **job description**
- Detect **matched skills**
- Identify **missing skills**
- Calculate **similarity score**
- Provide **alignment category**
- Interactive **Streamlit web interface**


## Tech Stack

- **Python**
- **Streamlit** – UI framework
- **pdfplumber** – resume text extraction
- **NLTK** – text preprocessing
- **scikit-learn**
  - TF-IDF Vectorizer
  - Cosine Similarity
